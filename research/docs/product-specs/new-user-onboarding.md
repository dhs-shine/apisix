# New User Onboarding for APISIX Research

## 이 연구 디렉토리를 처음 읽는 사람에게

APISIX는 다음 순서로 이해하면 가장 빠르다.

1. 제품이 무엇인지 이해한다
2. 런타임이 어떻게 부팅되는지 본다
3. 배포 모드를 구분한다
4. 플러그인이 어디서 실행되는지 이해한다
5. 테스트가 어떻게 동작을 보증하는지 확인한다
6. 운영 신호가 어디서 노출되는지 확인한다

## 핵심 개념

- APISIX는 설정 주도형 시스템이다.
- 동작의 상당 부분은 플러그인 조합에서 나온다.
- 테스트 파일은 실행 가능한 예제다.
- observability와 security는 핵심 제품 표면이다.
