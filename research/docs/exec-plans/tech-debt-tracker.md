# Research Tech Debt Tracker

## 목적

이 문서는 코드 부채가 아니라 연구 부채를 추적한다. 즉, APISIX를 agent-first 관점에서 다루기 위해 추가 조사나 문서화가 필요한 영역을 기록한다.

## 항목

| 상태 | 항목 | 설명 |
| --- | --- | --- |
| open | 플러그인 카탈로그 요약 부족 | `apisix/plugins/` 전체를 계층별로 요약한 맵이 아직 없음 |
| open | 테스트 분류 인덱스 부족 | `t/` 하위 테스트를 기능군별로 묶은 연구 문서가 아직 없음 |
| open | CI 워크플로 전체 맵 부족 | `ci/` 스크립트와 GitHub Actions 역할 연결 문서가 아직 없음 |
| open | AI Gateway 기능군 전용 조사 부족 | 최신 AI 관련 플러그인과 MCP bridge 분석이 별도 문서로 정리되지 않음 |
| open | 운영 시그널 매핑 부족 | 로그, 메트릭, trace가 어떤 플러그인과 연결되는지 일람이 없음 |
